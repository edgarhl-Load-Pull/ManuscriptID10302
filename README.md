# ManuscriptID10302
This repository contains MATLAB scripts and datasets used to reproduce the figures presented in the paper:

"Comparison of Two Harmonic Load-Pull Systems Using Frequency-Domain and Time-Domain Measurements for Evaluating GaN Transistor Output Power and Efficiency"

## Requirements
MATLAB R2020 or later.

## Reproducing Figures

Run the scripts:

- reproduce_H2_sweep_Popt.m
- reproduce_H2_sweep_Eopt.m

These scripts will generate the figures comparing:

• VNA-based harmonic load-pull  
• Oscilloscope-based harmonic load-pull  

## Data files

The `.mat` files contain processed experimental data used to generate the figures.
